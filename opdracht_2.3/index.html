Onderzoek

Waarom noem je de eerste HTML-pagina; index.html?

Omdat dit de eerste pagina is, dus de homepagina. Hierdoor weet de server precies
welke pagina zij als eerst moet openen. Dit maakt het een stuk makkelijker voor de developer
zodat hij dit niet zelf hoeft aan te geven.

Met welke HTML versie werken we nu?

HTML5, omdat dit de "Nieuwste" versie is. Daarom is er ook veel
infomatie op het Internet beschikbaar over deze versie. Zeer handig voor studenten!

Met welke CSS versie werken we nu?

CSS3, omdat dit de laatse versie van CSS is (in 1999 gepubliceerd).

Hoe kan het dan een website er net iets anders uit ziet in een andere browser? (Edge, Chrome, Firefox, Opera)

Omdat verschillende webbrowsers de websites anders inladen.
Ze zijn ook in verschillende talen geschreven of op een andere manier.

Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

Chrome. Chrome gebruikt met 16GB, 4% ram. Hoe meer tabs je opent, hoe hoger dit wordt.
